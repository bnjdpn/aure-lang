# Changelog

## 1.0.0 - 2026-06-22

- Promote Aure to 1.0.0 with a stable small-language core.
- Add assignment expressions for existing bindings.
- Add indexed assignment for arrays and maps.
- Add `for` loops over arrays, strings, and map keys.
- Add `break` and `continue` for loop control.
- Add short-circuiting `and` and `or` expressions.
- Add collection built-ins: `assert`, `push`, `pop`, `keys`, `values`, and `contains`.
- Add a runnable collections example and update documentation for 1.0 behavior.

## 0.1.0 - 2026-06-22

- Introduce the Aure language and Python reference interpreter.
- Add lexer, parser, runtime, closures, arrays, maps, loops, conditionals, and pipelines.
- Add CLI support for files, inline evaluation, REPL, and version output.
- Add examples, language guide, release notes, contribution guide, and tests.
